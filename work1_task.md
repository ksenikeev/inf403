
## Задание для самостоятельной работы
- Написать класс состоящий из 3-х элементов одного типа
- В классе должен быть конструктор, принимающий в качестве аргумента массив из элементов
этого типа. Элементы массива инициализируют члены класса:
0-й элемент 1-й член
1-й элемент 2-й член
2-й элемент 3-й член
- Если инициализирующий массив имеет неподходящий размер - бросать исключение.
- В классе должен быть один метод, возвращающий элемент по индексу (от 0 до 2):
  0 -> 1-й член
  1 -> 2-й член
  2 -> 3-й член
если индекс выходит за границы - бросать исключение.

Протестировать класс на типах String и Integer
Использовать дженерики.
